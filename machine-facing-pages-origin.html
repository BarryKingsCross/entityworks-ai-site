Origin and Scope Statement

This publication defines and names a previously unarticulated representational risk surface arising from the use of undisclosed machine-facing digital material. Prior to this work, no shared terminology, reference framework, or disclosure construct existed to describe the governance implications of machine-targeted publishing surfaces that fall outside an organisation’s human-facing navigation or perceptual footprint.

EntityWorks introduces the terms Machine-Facing Pages and Machine-Facing Page Declaration (MFPD) as part of the EntityWorks Standard in order to provide a neutral, descriptive reference for identifying, discussing, and disclosing this class of material. This work does not advocate adoption, mandate behaviour, or imply enforcement. Its purpose is solely to establish definitional clarity and a stable point of reference for future discussion, analysis, and oversight.

The formal definition and analysis of this concept is set out in the following publication:
